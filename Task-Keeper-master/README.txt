В приложении реализовано хранение через coredata
- удаление задачи (свайп справа налево в таблице)
- добавление задачи (время, текст задачи, тип задачи)
- отметка задачи как выполненную (свайп в слева направо)
- сортировка автоматическая в тейблвью по типу задачи (типы - разные цвета тайтла задачи)
- смена темы на 3 возможных (с анимацией ахахах)
